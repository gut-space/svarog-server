SVAROG Project

This is a small network of satellite ground stations, which aims to receive signals from satellites. It's a project created in 2020 by Sławek Figiel, Tomek Mrugalski and Ewelina Omernik, three students of Space and Satellite Technologies studies of Gdańsk University of Technology.

This project is available on github: server and station.
